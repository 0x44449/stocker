# 작업: analyzer 프로젝트 초기 구조 생성

## 목표

Python 분석 엔진 프로젝트 초기 구조 생성

## 범위

**할 것:**
- Poetry 프로젝트 생성
- 의존성 추가
- FastAPI 기본 구조
- 헬스체크 엔드포인트 (서버 동작 확인용)

**안 할 것:**
- 실제 분석 로직 구현
- DB 연결
- LLM 호출

## 구조

```
apps/analyzer/
  ├─ pyproject.toml
  ├─ README.md
  ├─ analyzer/
  │    ├─ __init__.py
  │    └─ main.py         ← FastAPI 앱
  └─ tests/

```

## 의존성

```
fastapi
uvicorn
langchain
langchain-ollama
sentence-transformers
psycopg2-binary
pgvector
```

## 실행 방법

```bash
cd apps/analyzer
poetry install
poetry run uvicorn analyzer.main:app --reload
```

## main.py 내용

```python
from fastapi import FastAPI

app = FastAPI(title="Stocker Analyzer")

@app.get("/health")
def health():
    return {"status": "ok"}
```
