# StockMasterCrawlController Swagger 응답 정보 추가

## 배경

- 현재 `Map<String, String>` 반환해서 Swagger에서 실제 응답 필드를 알 수 없음
- NewsCrawlController는 record DTO 사용 중

## 목표

StockMasterCrawlController에 Swagger 응답 정보 추가

## 관련 파일

```
apps/api/src/main/java/com/hanzi/stocker/api/internal/StockMasterCrawlController.java
```

## 할 것

현재:
```java
public Map<String, String> trigger() {
    return Map.of("status", "started");
}
```

변경:
```java
public record TriggerResponse(String status) {}

public TriggerResponse trigger() {
    return new TriggerResponse("started");
}
```

## 완료 조건

- Swagger에서 응답 스키마 확인 가능
