# 작업: Stage 1 종목 추출 테이블 생성

## 목표

뉴스에서 기업명 추출 파이프라인(Stage 1)을 위한 테이블 생성

## 범위

**할 것:**
- news_company_extraction 테이블 생성 (상태 관리)
- news_company_extraction_result 테이블 생성 (추출 결과)
- Flyway 마이그레이션 파일 추가

**안 할 것:**
- 데이터 삽입
- Analyzer 코드 수정

## 테이블 설계

### news_company_extraction (상태 관리)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | BIGSERIAL | PK |
| news_id | BIGINT | FK → news_raw.id, UNIQUE |
| status | VARCHAR(20) | pending, processing, done, failed |
| created_at | TIMESTAMP | 생성 시각 |
| processed_at | TIMESTAMP | 처리 완료 시각 (nullable) |

### news_company_extraction_result (추출 결과)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | BIGSERIAL | PK |
| extraction_id | BIGINT | FK → news_company_extraction.id |
| company_name | VARCHAR(200) | 추출된 기업명 |

## 마이그레이션 파일

- 위치: `apps/api/src/main/resources/db/migration/V6__add_news_company_extraction.sql`

## 완료 조건

- 테이블 2개 생성 확인
