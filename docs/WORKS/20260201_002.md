# KRX 종목 마스터 크롤러 수동 트리거 엔드포인트

## 배경

- 종목 마스터 크롤러가 스케줄러로만 동작해서 수동 테스트/트리거 불가
- 뉴스 크롤러처럼 수동 트리거 엔드포인트 필요

## 목표

종목 마스터 크롤링을 수동으로 트리거하는 엔드포인트 추가

## 범위

**할 것:**
- POST /internal/crawl/stock-master - 수동 트리거 (비동기)

**안 할 것:**
- 중복 실행 방지 (950건이라 금방 끝남)
- 상태 조회 엔드포인트

## 엔드포인트

**POST /internal/crawl/stock-master**
- 종목 마스터 크롤링 트리거
- 비동기 실행 (트리거 후 즉시 응답)

응답:
```json
{ "status": "started" }
```

## 파일 구조

```
apps/api/src/main/java/com/hanzi/stocker/api/internal/
└── StockMasterCrawlController.java   # 신규
```

## 변경 필요

현재 구현된 경로:
```java
@RequestMapping("/internal/crawl/master")
```

변경:
```java
@RequestMapping("/internal/crawl/stock-master")
```

## 완료 조건

- POST /internal/crawl/stock-master 호출 시 크롤링 시작
