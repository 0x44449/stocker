# 작업: Analyzer DB 연결 및 뉴스 조회

## 목표

Analyzer에서 PostgreSQL 연결하고 미처리 뉴스 조회

## 범위

**할 것:**
- SQLAlchemy 의존성 추가
- DB 연결 설정 (database.py)
- 모델 정의 (models.py)
- 미처리 뉴스 조회 엔드포인트

**안 할 것:**
- LLM 호출
- 결과 저장

## 의존성 추가

```bash
poetry add sqlalchemy
```

## DB 연결 정보

```
host: localhost
port: 5432
database: stocker
user: stocker
password: stocker
```

## 구조

```
analyzer/
  ├─ main.py          ← FastAPI 앱, 엔드포인트
  ├─ database.py      ← 엔진, 세션 설정
  └─ models.py        ← 테이블 모델
```

## 모델

### NewsRaw

| 컬럼 | 타입 |
|------|------|
| id | BigInteger, PK |
| title | String |
| raw_text | String |

### NewsCompanyExtraction

| 컬럼 | 타입 |
|------|------|
| id | BigInteger, PK |
| news_id | BigInteger, FK |

(미처리 뉴스 조회용으로 최소한만)

## 엔드포인트

```
GET /pending-news?limit=10
→ news_company_extraction에 없는 뉴스 목록 반환 (id, title)
```

## 완료 조건

- `poetry run uvicorn analyzer.main:app --reload` 실행
- `GET /pending-news` 호출 시 미처리 뉴스 목록 반환
