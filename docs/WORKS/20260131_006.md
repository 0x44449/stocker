# 한국경제(hk) 프로바이더 sitemap URL 수정

## 배경

- 뉴스 크롤링 시 hk 프로바이더에서 기사 URL이 0건 수집됨
- 원인: HkProvider에 설정된 sitemap URL이 모두 404
  - `sitemap/sitemap-news.xml` → 404
  - `sitemap.xml` → 404
- robots.txt 확인 결과 실제 sitemap 위치 확인됨

## 목표

HkProvider의 sitemapHints를 실제 유효한 URL로 수정

## 관련 파일

```
apps/api/src/main/java/com/hanzi/stocker/ingest/news/provider/hk/HkProvider.java
```

## 할 것

sitemapHints 반환값 수정:

현재:
```java
return List.of(
        BASE_URL + "/sitemap/sitemap-news.xml",
        BASE_URL + "/sitemap.xml"
);
```

변경:
```java
return List.of(
        BASE_URL + "/sitemap/latest-article.xml",
        BASE_URL + "/sitemap/daily-article.xml"
);
```

## 안 할 것

- 다른 프로바이더 수정
- 파서 로직 변경
