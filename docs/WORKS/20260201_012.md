# DB 마이그레이션 - extracted_name nullable 변경

## 배경

- 이전 세션에서 extracted_name을 nullable로 하기로 결정
- 이유: LLM이 추출하지 않았지만 사람이 수동으로 매핑 추가하는 경우
- extracted_name이 NULL이면 → 사람이 직접 추가한 매핑 (학습 데이터 구분용)

## 현재 상태

```sql
extracted_name VARCHAR(200) NOT NULL
```

## 변경 후

```sql
extracted_name VARCHAR(200)  -- nullable
```
